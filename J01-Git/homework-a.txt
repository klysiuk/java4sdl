Домашнее задание по лекции J01-Git.

Задания необходимо сделать до практических занятий. 
Имейте это ввиду, зная, что есть группа в четверг.

(1 point) 
1. Обновить профиль в github. Напишите, пожалуйста, в Account -> Name -> сначала фамилию, затем имя (i.e. Lozenko Stanislav) 

(2 points)
2. Необходимо закоммитить ваш проект "HelloWorld" из первой лекции в ваш собственный проект персонально у вас на гитхабе. 

(1 point)
3. Написать ессе, в котором вы сравниваете git с любой другой системой версии контроля. 
Прошу воздержаться от тупого копипаста и реально вникнуть в суть.

(5 points)
* (усложнено)
4. Реализовать 3 проекта HelloWorld, в каждом из которой используется разные branch стратегии.
Три одинаковых helloworld - три разных branch стратегии. Презентация лежит в этой же папке.

(2 points)
* (усложнено)
5. Реализовать проект helloworld-svn с помощью другой версии контроля - Subversion (SVN).
Результатом проекта будет ваш helloworld + специально закоммиченные файлы .svn

Задание с звездочкой необязательно, но всегда помните - усложненные задания помогут вам разобраться лучше в материале.
Не ленитесь.
